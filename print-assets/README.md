# Print Assets

ใช้ไฟล์รูปในโฟลเดอร์นี้แทนรูปเดิมใน Google Sheet แม่แบบ เพื่อกันปัญหา PNG พื้นโปร่งใสกลายเป็นสีดำตอน export PDF

- `1-white.jpg` ใช้แทนแถบข้อความหัวกระดาษยาว
- `2-white.jpg` ใช้แทนโลโก้กรมควบคุมโรค
- `3-white.jpg` ใช้แทน QR Code
- `4-white.jpg` ใช้แทนแถบข้อความส่วนท้าย

คำแนะนำ:

1. ลบรูปเดิมในชีต `ใบแจ้งอัตราค่าบริการ` และ `ใบแจ้งค่าบริการ`
2. แทรกรูปใหม่แบบ `Insert image over cells`
3. วางตำแหน่งและขนาดให้ใกล้เคียงของเดิม
4. ทดสอบ export PDF อีกครั้ง
